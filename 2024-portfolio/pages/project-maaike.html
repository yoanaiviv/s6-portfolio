Maaike Girardin

The website is a personal portfolio for a Dutch singer and music teacher from Tilburg.

This is a group project that was made for Maaike Girardin as part of my studies. We created the prototype using Adobe XD. The website is one pager showing her work and songs.

When compiling the application we used HTML5, CSS, Vanilla JavaScript.
