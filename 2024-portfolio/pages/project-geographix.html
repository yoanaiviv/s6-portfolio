GeoGraphix

GeoGraphix is a documentational website for the design system of Geodan.

This is an individual project that was made for Geodan as part of my internship. I did a throughout research about miscommunication regarding the design system in the company. I created a high-fidelity prototype using Figma.

When compiling the application I used HTML5, CSS, Angular CLI and PrimeNG components. It was my first time using Angular CLI and also using premade component.I used this framework simply because the company is using it.

Designing dreams, one pixel at a time.
